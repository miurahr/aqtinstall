Security Policy
===============

Supported Versions
------------------

+---------+---------------------+
| Version | Status              |
+=========+=====================+
| 2.1.x   | Stable version      |
+---------+---------------------+
| 2.0.x   | Security fixes only |

| < 2.0   | not supported       |
